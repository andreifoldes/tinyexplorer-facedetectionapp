Installation¶

macOS Requirements¶

Before installing the Face Detection App on macOS, ensure your system meets these requirements:

  • macOS Version: macOS 10.15 (Catalina) or later
  • Processor: Intel x64 or Apple Silicon (M1/M2/M3)
  • Memory: Minimum 8GB RAM (16GB recommended for video processing)
  • Storage: At least 2GB free disk space for the application and models

Installing on macOS¶

To install the Face Detection App on your Mac:

  1. Download the Application
  2. Visit our GitHub releases page

  3. Download the .dmg file for your Mac architecture:

    • TinyExplorer-FaceDetectionApp-mac-x64.dmg for Intel Macs
    • TinyExplorer-FaceDetectionApp-mac-arm64.dmg for Apple Silicon Macs (M1/M2/M3)

  4. Install the Application

  5. Double-click the downloaded .dmg file to mount it
  6. Drag the "TinyExplorer FaceDetectionApp" from the mounted disk image to your Applications folder

  7. Eject the disk image when copying is complete

  8. Remove macOS Quarantine (Required)

Why is this necessary?

macOS applies a "quarantine" attribute to applications downloaded from the internet as a security measure. This prevents unsigned or unnotarized applications from running and protects users from potentially malicious software. Since our Face Detection App is not signed with an Apple Developer certificate, macOS will block it from launching.

How to unquarantine the app:

Method 1: Using Terminal (Recommended) bash xattr -cr "/Applications/TinyExplorer FaceDetectionApp.app"

Steps: - Open Terminal (found in Applications â†’ Utilities) - Copy and paste the command above - Press Enter to execute - The app is now unquarantined and safe to launch

Method 2: Using System Settings (if Terminal method doesn't work) - Try to launch the app first (it will be blocked) - Go to System Settings â†’ Privacy & Security (or System Preferences â†’ Security & Privacy on older macOS) - Look for a message about the blocked app - Click "Open Anyway" or "Allow" - Confirm by clicking "Open" in the dialog that appears

Method 3: Right-click Override - Right-click on the app in Applications folder - Select "Open" from the context menu - Click "Open" in the warning dialog that appears - This bypasses the initial quarantine check

  1. Verify Installation
  2. The app should now launch without security warnings
  3. You'll see the main Face Detection App interface
  4. The Python environment will initialize on first launch (this may take 30-60 seconds)

Launching the Application¶

To start the Face Detection App on macOS:

  1. Open Finder and navigate to your Applications folder
  2. Locate TinyExplorer FaceDetectionApp
  3. Double-click the app icon to launch
  4. The application will initialize the Python environment on first launch (this may take a few moments)
  5. The main interface will appear, ready for use

Quick Launch Tips¶

  • Spotlight Search: Press Cmd + Space and type "TinyExplorer" to quickly find and launch the app
  • Dock: Drag the app from Applications to your Dock for one-click access
  • Launchpad: Find the app in Launchpad for touch-friendly launching

Interface Overview¶

The Face Detection App user interface on macOS

The main interface consists of several key areas:

  • File/Folder Selection: Choose images or videos for processing
  • Click "Browse File" to select individual images/videos

  • Click "Browse Folder" for batch processing

  • Model Selection: Pick the face detection model to use

  • YOLO models (fast, general-purpose)

  • RetinaFace (accurate, better for challenging conditions)

  • Confidence Threshold: Adjust the detection sensitivity (0.0 - 1.0)

  • Lower values detect more faces but may include false positives

  • Higher values are more selective

  • Output Settings: Configure where results are saved

  • Choose output folder location

  • Results include detected faces and CSV reports

  • Start Button: Begin the detection process

  • Real-time progress updates during processing

  • Cancel option available during long operations

  • Results Display: View the output of the detection process

  • Summary statistics
  • "Open Results Folder" for quick access

Troubleshooting macOS Issues¶

If you encounter issues on macOS:

  1. "App is damaged and can't be opened"

  2. Open Terminal and run: xattr -cr /Applications/TinyExplorer\ FaceDetectionApp.app

  3. Python environment not loading

  4. Ensure you have sufficient disk space

  5. Try reinstalling the application

  6. Slow performance on older Macs

  7. Reduce batch size for folder processing
  8. Use lighter models (YOLOv8n instead of YOLOv8l)
